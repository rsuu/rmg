# rmg

Mnaga Reader

## Install

+ Dependency
    + sdl2
    + Optional Dependency
        + tar
        + zip

### Build

```bash
cargo build --release
```

## Usage


```bash
cargo run --release -- --size 900,800 tests/files/img.tar

# OR

rmg --size 900,900 tests/files/img.tar
```
